A classic Snake Game built using HTML, CSS, and JavaScript. Control the snake, eat food to increase your score, and avoid collisions with the walls or your own body.

🎮 Features
Classic snake gameplay
Keyboard controls using arrow keys
Random food generation
Score tracking
Game Over detection
Responsive and simple UI
Start game on user input
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

snake-game/
│── index.html
│── style.css

└── README.md
🚀 How to Run
Clone the repository:
git clone https://github.com/your-username/snake-game.git
Navigate to the project folder:
cd snake-game
Open index.html in your browser.

No additional dependencies or installations are required.

🎯 How to Play
Press any arrow key to start the game.
Use the Arrow Keys to move the snake:
⬆️ Up
⬇️ Down
⬅️ Left
➡️ Right
Eat the food to grow longer and increase your score.
The game ends if:
The snake hits the wall.
The snake collides with itself.
